# Claude Context — AI Gallery Starter

> PROJECT TYPE: Frontend only — no backend, no database, no external services
> ห้ามนำ credentials, server, หรือ service ใดๆ จาก global CLAUDE.md มาใช้ใน project นี้

## Stack

- Vite + React + TypeScript + Tailwind CSS
- Static frontend เท่านั้น
- Deploy target: Hostinger Node.js Hosting (หรือ Coolify บน VPS)

## กฎสำหรับ Claude

- project นี้ไม่มี backend, ไม่มี database, ไม่มี auth
- ถ้าจะเพิ่ม feature ที่ต้องการ backend → บอก user ก่อนเสมอว่าต้องมีอะไรเพิ่ม
- ห้าม assume ว่ามี VPS, SSH, Pocketbase, หรือ service ใดๆ อยู่แล้ว
- งานหลัก: แก้ไข UI, design, layout, เนื้อหา — ตามที่ user สั่งเท่านั้น
